# Getting Started with Radiant System

Radiant System is a Story OS designed to orchestrate creative workflows.

## Prerequisites

- Node.js 18+
- pnpm
- OpenAI API Key (optional for AI modules)

## Installation

```bash
git clone https://github.com/radiant-system/radiant.git
cd radiant-system
pnpm install
```

## Quick Start

1. **Initialize the system:**
   ```bash
   pnpm dev
   ```

2. **Access the Web Interface:**
   Open [http://localhost:3000](http://localhost:3000)

3. **Explore the Manual:**
   Visit `/manual` to understand the build order and operational protocols.

## Core Workflows

- **Transformation:** Use DropFrame to refactor content.
- **Production:** Use Grindline for batch generation.
- **Learning:** Access the VSM School curriculum.
